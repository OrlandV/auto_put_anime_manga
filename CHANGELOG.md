### Версия 0.6.9 20251128
+ В модуле _константы_ (`constants`) добавлены следующие:
  + `WPES = 'https://en.wikipedia.org/wiki/'`,
  + `USER_AGENT = f'Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:{FIREFOX_VER}) Gecko/20100101 Firefox/{FIREFOX_VER}'`
    (указан браузер Mozilla Firefox в ОС Windows 10, версия которого указывается в модуле конфигурации).
+ В модуле _конфигурации_ (`config`)
  + добавлена версия Firefox для User-Agent: `FIREFOX_VER = '145.0.2'` (рекомендуется к актуализации);
  + к _WA-форматы anime в ANN_ добавлен `{'special': 'ТВ-спэшл'}`;
  + к _игнорируемым жанрам_ добавлены `история`, `Harem` и `Historical`;
  + к _жанрам AnimeNewsNetwork_ добавлен `{'magic': 'магия'}`.
+ При импортировании модуля _поиска страниц в **Wikipedia** (en) и их обработки_ (`wikipedia_`) происходит
  + переключение `API_URL` в пакете `wikipedia` на протокол `HTTPS`,
  + установка `User-Agent` в пакете `wikipedia` в значение из константы `USER_AGENT`.
+ Добавлен вывод в консоль _контрольной информации о текущей выполняемой функции_. (Когда обрабатывается франшиза,
  может показаться, что программа зависла (бесконечный цикл либо сетевые блокировки).)
+ В модуле _поиска страниц в **Wikipedia** (en) и их обработки_ выделены функции
  + _поиска инфо-бокса_ (`wikipedia_._pos_table`),
  + _поиска текста между HTML-тегами_ (`wikipedia_._pos_ab`).
+ Добавлена проверка наличия наименований **ANN** и **Wikipedia** (en) в **WA**: поиск и добавление отсутствующих.
  А также извлечение данных из вновь найденных страниц.
  + Выделена функция _обновления словарей HTML-страниц anime и манги из **WA** текущими данными_. (`wa_pages_update`)

* _Полученный HTML-код страницы в **WA**_ теперь сразу проходит декодирование символов. (`world_art.html`)
* _Декодирование символов_ теперь происходит не только Юникод, но и шестьнадцатиричных кодов. (`decode_name.decode_name`)
* Поиск связей anime и манги в **WA** теперь учитывает ведущий ноль в номере в номерованном списке.
  (`world_art.anime_pages`, `world_art.manga_pages`, `world_art.manga_pages_from_anime`, `world_art.related_anime`)
* _Нормализация наименования_ теперь производит
  * декодирование символов и
  * замену дефисов пробелами.

  (`decode_name.normal_name`)
* _Первичный поиск XML-страниц манги и anime в **ANN**_ теперь отфильтровывает ссылки на XML-страницы новелл.
  (`animenewsnetwork.search_pages`)
* _Поиск и получение XML-страниц манги и anime в **ANN** на базе уже полученных XML-страниц_ теперь
  * больше не ищет и не загружает XML-страницы повторно,
  * обрабатывает случаи отсутствия XML-страниц anime в БД **ANN** (`no result`).

  (`animenewsnetwork.pages`)
* _Первичный поиск данных по манге в **MangaUpdates**_ теперь обрабатывает ошибку подключения: в случае возникновения
  выдаёт уведомление в консоль и возвращает `None`. (`mangaupdates.search_pages`)
* _Получение HTML-кода страницы по внутренней ссылке в **Wikipedia** (en)_ теперь
  * происходит по протоколу `HTTPS` с указанием `User-Agent` в заголовке запроса, который указывается в модуле констант
    (`constants.USER_AGENT`) и рекомендуется к актуализации;
  * результат декодируется.

  (`wikipedia_.page`)
* Функция _получения страницы (HTML-кода) из **Wikipedia** (en)_ заменена классом _страницы из **Wikipedia** (en)_.
  При инициализации объекта происходит
  1. поисковый запрос (`wikipedia.search`),
  2. поиск искомого наименования среди результатов поискового запроса,
  3. получение страницы по выбранному результату поискового запроса (`wikipedia.WikipediaPage`),
  4. присваивание свойству `html` декодированного HTML-кода страницы,
  5. присваивание свойству `url` фрагмента URL-адреса (после `/wiki/`), содержащего заголовок страницы.

  Если среди результатов поискового запроса искомого наименования нет, тогда происходит переключение на функцию `page`.
  Если сервер **Wikipedia** (en) сообщает об ошибке (`wikipedia.WikipediaException`) 15 раз подряд (около 15 секунд),
  также происходит переключение на функцию `page`. Результат выполнения функции `page` присваивается свойству `html`,
  а её параметр — свойству `url`. (`wikipedia_.Page`)
* _Поиск страниц по наименованию в **Wikipedia** (en)_ полностью переписан.
  * Вторым параметром функция принимает необязательный дополняемый словарь `{title: HTML}` страниц в **Wikipedia** (en).
    В этот словарь дописываются найденные страницы, после чего он возвращается. (Цель такой схемы — устранение
    повторного поиска и загрузки.)
  * В качестве ключей результирующего словаря прописываются заголовки страниц из `Page.url`.
  * В инфо-боксе обрабатываются списки `Related series`, `Feature films` и `Related works`. 

  (`wikipedia_.search_pages`)
* _Извлечение из страницы частей страницы (частей инфо-блоков) по отдельным манге и anime в **Wikipedia** (en)_ почти
  полностью переписано. (`wikipedia_.manga_anime_in_page`)
* Оптимизирован код _извлечения даты премьеры из части страницы в **Wikipedia** (en)_. (`wikipedia_.date_of_premiere`)
* _Извлечение оригинального наименования из страницы в **WA**_ теперь не декодирует результат. (`world_art.title_orig`)
* _Извлечение ромадзи наименования из страницы в **WA**_ теперь не декодирует результат. (`world_art.title_rom`)
* _Извлечение английского наименования из страницы в **WA**_ теперь не декодирует результат. (`world_art.title_eng`)
* _Извлечение русского наименования из страницы в **WA**_ теперь не декодирует результат. (`world_art.title_rus`)
* _Извлечение имён персоны из страницы в **WA**_ теперь не декодирует оригинальное имя. (`world_art.people`)
* _Извлечение наименования из XML-страницы в **ANN**_ почти полностью переписано. (`animenewsnetwork.title`)
* _Извлечение авторов манги или режиссёров anime из XML-страницы в **ANN**_ теперь учитывает блоки
  `<div id="infotype-13">` и `<div id="infotype-12">`. (`animenewsnetwork.authors`)
* _Извлечение издания манги из XML-страницы в **ANN**_ теперь учитывает блок `<div id="infotype-2">`.
  (`animenewsnetwork.publication`)
* После _извлечения данных anime из XML-страницы в **ANN**_ в ромадзи наименование прописывается английское
  наименование, если ромадзи наименование совпадает с оригинальным наименованием и отличается от английского
  наименования. (`animenewsnetwork.extraction_anime`)
* _Извлечение формата anime из соответствующей части инфо-блока в **Wikipedia** (en)_ теперь не определив формата,
  но обнаружив в наименовании слова `movie` (регистр не важен), возвращает `FORM_WP['Anime film series']` (соответствует
  WA-формату `полнометражный фильм`). (`wikipedia_.anime_format`)
* _Извлечение количества эпизодов anime из соответствующей части инфо-блока в **Wikipedia** (en)_ помимо поля `Episodes`
  теперь учитывается поле `<abbr title="Number">No.</abbr> of episodes`. (`wikipedia_.number_of_episodes`)
* _Извлечение студий anime из соответствующей части инфо-блока в **Wikipedia** (en)_ теперь обрабатывает список студий,
  оформленный только тегом `<br />`. (`wikipedia_.studios`)
* _Извлечение имён авторов манги или режиссёров anime из соответствующей части инфо-блока в **Wikipedia** (en)_ почти
  полностью переписано. (`wikipedia_.authors`)
* _Извлечение наименований издательства и издания в **Wikipedia** (en)_ теперь не добавляет в результирующий список
  издание, если оно там уже имеется. (`wikipedia_.publications`)
* В модуле _поиска страниц в **MangaUpdates** и их обработки_ _выбор наименования манги через обращение к пользователю_
  теперь уточняет у пользователя и ромадзи наименование, а не берёт его из заголовка (`'title'`) в JSON-ответе.
  (`mangaupdates.select_title`)
* При _формировании единого словаря данных из словарей данных, извлечённых из страниц_ (`data_join`) теперь
  * обрабатываются числовые ключи, взятые из JSON-словаря в строковом формате;
  * в единый словарь данных записываются ID манги в **WA** и **ANN**;
  * поиск соответствующей манги или anime в **ANN**
    * в первую очередь производится по ID, затем по дате премьеры (более детально), ромадзи наименованию и авторам;
    * при обработке найденной манги в единый словарь данных записывается ID манги в **ANN**, а поиск прекращается;
  * в _сверке имён (ромадзи и оригинальных) авторов манги_ (`data_join.maut`) в качестве параметра принимается список
    словарей имён авторов сравниваемой манги;
  * в _поиске соответствующей манги в словаре **Wikipedia** (en)/**MangaUpdates** и дополнении недостающими данными
    единого словаря данных по манге_ (`data_join.mwp`/`data_join.mmu`)
    * _даты премьеры_ сравниваются более детально, а также сравниваются _наименования_, а найдя — поиск не продолжается;
    * _издания_ добавляются только в пустой список;
  * величина погрешности в указании года премьеры манги в **MangaUpdates** относительно **ANN** равна `1`;
  * в _поиске соответствующего anime в словаре **Wikipedia** (en) и дополнении недостающими данными единого словаря
    данных по anime_ (`data_join.awp`)
    * исправлена ошибка отсутствия _продолжительности_ anime в едином словаре данных;
    * если _даты премьеры_ или _режиссёры_ не совпадают, сравниваются _наименования_ и _годы_ из дат премьер;
    * если найден _режиссёр_, отсутствующий в едином словаре данных, перед его добавлением проверяется наличие русского
      имени и, при отсутствии, выполняется запрос в WA по ромадзи имени;
    * найдя соответствующее anime, поиск не продолжается;
  * в _замене None в обязательных полях на значения по умолчанию (или заглушки)_ (`data_join.not_none`) добавлена
    заглушка `—<Нет информации>—` для списка _студий_;
  * поиск в едином словаре данных манги, связанной с anime, производится по ID в соответствующей БД
    (**WA** или **ANN**);
  * исправлена ошибка отсутствия _студии_ anime в **ANN**.

### Версия 0.5.0 20250113
+ Доработан поиск манги в инфобоксе страницы в **Wikipedia** (en).
+ Заменён алгоритм извлечения манги из **ANN** (обработка глобального словаря).
    + Расширен поиск манги в **MangaUpdates**. В случае неудачного поиска возвращается `None`.
    + Ромадзи наименование извлекается из **MangaUpdates**, либо из **ANN**.
    + Поиск в **Wikipedia** (en) производится по извлечённому ромадзи наименованию,
      либо по наименованию из входных данных.
    + Если манга найдена в **MangaUpdates**, производится попытка извлечения изданий манги из **MangaUpdates**.
    + Если в **MangaUpdates** манга не найдена, то производится попытка извлечения изданий манги из **Wikipedia** (en).
    + Издания из **Wikipedia** (en) теперь извлекаются списком.
    + Если издание в **Wikipedia** (en) не найдено, то оно извлекается из **ANN** вместе с датой премьеры.
    + Если манга найдена в **MangaUpdates**, производится попытка извлечения жанров из **MangaUpdates**.
      Иначе жанры извлекаются из **ANN**.
    + В файл конфигураций добавлен словарь жанров **ANN** `GENRES_ANN`.
    + Если жанры не найдены, пользователю предлагается выбрать их из БД.
    + Если из **ANN** дата не извлечена, то она извлекается из **Wikipedia** (en).
    + Проверяется извлечённая дата. Если её нет или оканчивается на `-12-31`, то в примечании отмечается отсутствие
      даты премьеры.
    + Если дата так и не найдена, прописывается «заглушка» `1900-01-01`.
    + Английское наименование извлекается из **ANN**, либо предоставляется пользователю выбор из списка,
      полученного из **MangaUpdates**.
    + Количество томов извлекается из **ANN**, либо из **MangaUpdates** (если манга найдена в **MangaUpdates**),
      либо прописывается `1` и в примечании отмечается отсутствие информации о количестве томов.
    + Из **ANN** извлекаются авторы манги.
    + Из **ANN** извлекается оригинальное наименование.
    + Формируется строка примечания.
    + Если оригинальное наименование пусто (не найдено), то в него прописывается ромадзи.
+ При извлечении манги из **MangaUpdates** (обработка глобального списка) извлекается либо количество томов,
  либо количество глав (когда том ещё не издавался). Во втором случае в примечании выключается указание о главах.
* Исправлено извлечение даты премьеры манги из **Wikipedia** (en), когда указаны месяц и год без числа (дня).
* Извлечение оригинального наименования из **WA** дополнено проверкой поля `Названия (яп.)`.
* Исправлено извлечение наименования из **ANN** и добавлено извлечение ромадзи наименования из заголовка.
* При извлечении anime из **WA**, если оригинальное наименование совпало с русским (не нашлось оригинальное),
  а ромадзи не пусто (нашлось), тогда в оригинальное наименование прописывается ромадзи.
* При извлечении из **WA**, если не найдено английское наименование (ни в WA, ни в ANN),
  а оригинальное совпадает с ромадзи, тогда в английское наименование прописывается ромадзи.
